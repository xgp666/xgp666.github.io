一次 Web 资源请求的具体过程

  • 客户端在 Web 浏览器输入需要访问的地址
  • Web 浏览器会请求 DNS 服务器，查询解析到指定域名和 Web 服务器的地址
  • 客户端与请求的 Web 服务器端建立连接（TCP 三次握手）
  • TCP 建立成功之后，发起 HTTP 请求
  • 服务器端收到客户端 HTTP 请求之后，会处理该请求
  • 处理客户端指定请求的资源
  • 服务器构建响应报文，响应给客户端
  • 服务器端将此信息记录到日志中
